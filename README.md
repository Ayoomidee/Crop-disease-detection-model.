# Crop-disease-detection-model.
A Deep Learning Project
Setup Instructions
1. Prerequisites
Ensure you have Python 3.12+ installed.
2. Clone the Repository
3. Install Dependencies
Install all necessary libraries using the provided requirements:
Dependencies include numpy, opencv-python, scikit-learn, matplotlib, and seaborn.
4. Prepare the Dataset
i. Download the PlantVillage Color Dataset.
ii. Update the data_path variable to point to your local directory.
5. Execution Flow
Run the full pipeline in order:
Step 1: Train the model and evaluate
Step 2: Generate performance visuals
